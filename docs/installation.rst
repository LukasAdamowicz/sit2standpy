pysit2stand Installation
=======================================

Requirements
---------------

- Python >= 3.7
- Numpy >= 1.16.2
- Scipy >= 1.2.1
- pywavelets >= 1.0.3

These are the versions developed on, and some backwards compatability may be possible.

Installation
------------

Run in the command line/terminal:

.. code-block::
   :linenos:

   pip install pysit2stand
